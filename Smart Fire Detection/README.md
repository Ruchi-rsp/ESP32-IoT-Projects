Smart Fire Detection System using ESP32
Overview
This project focuses on building an IoT-based fire detection system using ESP32 to provide early
fire alerts and reduce potential damage. The system continuously monitors environmental conditions 
and sends real-time notifications to the user.

Objective
To detect fire hazards at an early stage  
To send instant alerts to users remotely  
To improve safety using IoT-based monitoring  

 Technologies Used
- ESP32 Microcontroller  
- Flame Sensor / Smoke Sensor  
- Blynk IoT Platform  
- Embedded C / Arduino IDE  

Working Principle
- Sensors continuously monitor temperature/smoke/flame  
- ESP32 processes sensor data  
- If fire is detected:
   Alert is triggered  
   Notification is sent via Blynk app  
- User receives real-time updates on mobile  

 Features
- Real-time fire detection  
- Instant mobile notifications  
- Remote monitoring using IoT  
- Low-cost and efficient system
